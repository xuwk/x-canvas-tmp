Shadow Class

Defined in: Shadow:3
Module: xc.createjs.easeljs

This class encapsulates the properties required to define a shadow to apply to a DisplayObject via it's shadow property.

Example

myImage.shadow = new Shadow("#000000", 5, 5, 10);
